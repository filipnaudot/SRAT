# SRAT
SRAT is a simple remote acces tool capable of remote command execution aswell as file upload and download.


### Commands
| command              | Description                           |
| -----------          | -----------                           |
| get [filename]       | Download a file from remote           |
| put [filename]       | Upload a file to remote               |
